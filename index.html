EdTrack

View your progress! Track your future!

EdTrack is a versatile performance tracking tool that alleviates the burden on overworked teachers, oversized classrooms, and students to better track and learn from their performance. There will be three different account levels: manager, instructor, and student. The student will be able to log in which will account for attendence, turn in their assignments, and see their in class progress. Managers will be able to view the class progress, student progress, and associate it with the instructor. Possible functionality can be to view all feedback and job search.

Developed a fully responsive web application using javascript, redux, react, node.js that helps institutions track student performance and programmatically match students with appropriate mentors.

Main Features and Technologies: instructor schemas, routes and front end using Sequelize ORM, Postgres, Redux and React-Router. Developed an interactive user interface to upload user profile picture and files using Multer library, FileInput and Material Design. You can test the app at electrone901.com or you can view at invision.com
